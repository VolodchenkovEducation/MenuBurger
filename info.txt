Запуск чистого проэкта:
1. Удаляем папку .git, если она есть
2. В файле package.json удаляем привязку к GitHub
3. Открываем проэкт в VS Code
4. Запускаем коммандную строку из данной папки (bush)
5. Устанавливаем необходимые пакеты: npm i
6. Запускаем необходимый режим, скорре всего будет: npm run dev

Если нужно сформировать SVGСпрайт (собрать все иконки в 1 файл) из .SVG файлов:
1. поместить все файлы с расширением .SVG в папку D:\Desktop\NewProject\src\svgicons
2. зайти к командную строку VSCode (bush)  и выполнить команду npm run svgSprive

Запуск различных режимов:
1. Разработчика: npm run dev
2. ПостПродакшн: npm run build
3. Создание архива с готовым проэктом: npm run zip
4. Выгрузка результата на FTP сервер: npm run deploy 
	- gulp/cohfig/ftp.js - Нужно внести данные сервера
	- gulp/cohfig/path.js - Указать имя папки на сервере
	- gulp/gulpfile.js - Раскомментировать настройки FTP
